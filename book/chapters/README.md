# Clean Chapter Files

This folder will hold the cleaned publication chapters for **From the Storm to the Fire: The Truth Behind the Silence**.

## Naming Pattern

Use this format:

```text
chapter-01-the-storm-outside.md
chapter-02-the-fire-ignites.md
chapter-03-pain-takes-root.md
```

## Chapter File Rule

Each chapter file should contain only one clean chapter.

Do not include:

- alternate draft versions,
- AI/editor notes,
- duplicate sections,
- accidental table-of-contents fragments,
- unrelated appendices,
- repeated final thoughts.

## Heading Rule

Use:

```markdown
# Chapter 1 — The Storm Outside
```

or, when a part divider is needed:

```markdown
# Part I — The Storm

## Chapter 1 — The Storm Outside
```

Do not use multiple top-level book titles inside chapter files.
