# Source Manuscript Folder

This folder is for preserving the raw manuscript before publication cleanup.

## Rule

The top-level `README.md` currently holds the raw manuscript source. Before restructuring the book, copy that source here as:

```text
book/source/raw-manuscript.md
```

Do not delete or overwrite the original manuscript until this protected copy exists and has been verified.

## Why This Exists

The raw manuscript contains the full emotional record. The publication edition will be cleaner, but the raw source should stay protected so nothing important is lost during editing.
